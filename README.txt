README =====

To run from command line
1. Install https://github.com/nunit/nunit-console/releases/download/v3.11.1/NUnit.Console-3.11.1.msi
2. cd [to the folder where UITest.dll was built to]
3. "C:\Program Files (x86)\NUnit.org\nunit-console\nunit3-console.exe" UITest.dll

To run from solution
1. UITest's Debug properties
2. Start external program: C:\Program Files (x86)\NUnit.org\nunit-console\nunit3-console.exe
3. Command line arguments: https://stackoverflow.com/questions/9297992/debugging-tests-running-under-nunit

Make sure that the driver is for the correct browser version: https://chromedriver.chromium.org/ (I've updated the Chrome driver to v86 but IE and FireFox driver is still the version released in 2016)
